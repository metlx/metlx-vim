# metlx-vim
my config

## features
- leet color
- vim-go

### 1. install vundle 
```
git clone https://github.com/VundleVim/Vundle.vim.git ~/.vim/bundle/Vundle.vim
```
### 2. install plugins
```
:VundleInstall
```

### 3. install go-binaries
```
:GoInstallBinaries
```

### 4. (optional) update go-binaries
```
:GoUpdateBinaries
```
